QuadrantCard — a single finding in the 2x2 evidence matrix; the visual center of the report. Always pair evidence + rationale + an interview-focus prompt. Lay four out in a 2-column grid.

```jsx
<QuadrantCard
  quadrant="strong_with_evidence"
  title="React and TypeScript project ownership"
  source="GitHub repository shows a typed React/Vite app with component and API structure."
  rationale="Documented, consistent commit history on an original (non-fork) project."
  interviewFocus="Validate ownership, trade-offs and production depth."
/>
```

Quadrants: `strong_with_evidence`, `strong_needs_validation`, `weak_with_evidence`, `weak_needs_validation`.
