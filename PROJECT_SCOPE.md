# Project Scope

## Goal

Build a small, deployable MVP that demonstrates AI-native product engineering for technical recruiting.

The tool should analyze a candidate against a role using evidence from:

- job description;
- resume;
- LinkedIn export text/PDF;
- GitHub public repositories;
- portfolio links.

## Non-goals for the first version

- No LinkedIn login or cookie-based scraping.
- No final match score.
- No automatic hiring recommendation.
- No unsafe execution of candidate repositories.
- No full ATS replacement.
- No Loxo integration in the first MVP.

## First useful demo

A recruiter can:

1. Enter a job description and role constraints.
2. Upload or paste candidate evidence.
3. Run an analysis.
4. Review a four-quadrant evidence matrix.
5. Export STAR interview questions and a recruiter-facing summary.
