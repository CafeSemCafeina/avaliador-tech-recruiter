Card — the soft panel surface that holds nearly everything; use `default` for content, `sunken` for recessed groups, and the status tones for evidence quadrants.

```jsx
<Card tone="default" padding="md">…</Card>
<Card tone="confirmed">Strong with evidence</Card>
```

Tones: `default`, `sunken`, `confirmed`, `validate`, `gap`, `uncertain`. Padding: `none`/`sm`/`md`/`lg`.
