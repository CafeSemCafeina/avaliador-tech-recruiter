QualBadge — a labeled qualitative signal for the report header. Value is words, never a number or score. Lay several in a row/grid.

```jsx
<QualBadge label="Seniority Signal" value="Mid plausible — validate backend ownership" tone="validate" />
<QualBadge label="Stack Evidence" value="Strong in React/TS" tone="confirmed" />
<QualBadge label="Public Proof" value="Mixed" tone="uncertain" />
```
