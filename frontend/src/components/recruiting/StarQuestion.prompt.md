StarQuestion — a copy-friendly STAR interview question with optional follow-ups and a "good answer reveals" note. The Copy button puts the question + follow-ups on the clipboard.

```jsx
<StarQuestion
  index={1}
  question="Tell me about a project where you owned a feature from implementation to deployment…"
  followUps={["What trade-offs did you consider?", "How did you validate the solution?"]}
  reveals="ownership depth and production awareness"
/>
```
